
Module Overview

This document describes public-facing module concepts.

No proprietary implementation is included.

Reality Filter

Classifies input against reality-related dimensions:

domain
controllability
emotional tension
observation asymmetry
trust weighting
Intent Router

Routes input into broad intent categories:

normal
explanatory
architectural
strategic
planning
decision
relational
Observer Protocol

Handles incomplete or asymmetric information.

Core principle:

Observation is not always neutral.

Decode Policy

Prevents premature interpretation collapse.

It preserves ambiguity when the structure is unresolved.

Runtime Gate

Represents runtime-level capability boundaries.

Examples:

core runtime enabled / disabled
live fetch enabled / disabled
execution runtime enabled / disabled
mode escalation enabled / disabled
Bridge State

A compact runtime state representation.

Used to transfer structural state across stateless workflows.

F-Array

A flag array inside the bridge packet.

It encodes detected structural conditions and guardrail-relevant states.

Local Fallback

A degraded or intentional local-only mode.

Used when external models are unavailable or intentionally disabled.
