
Safety Boundary

Altora Core separates structural detection from action permission.

A detected signal is not execution permission.

A structural pattern is not prediction authority.

A runtime state is not an oracle.

Public Safety Principles
No hidden execution permission
No automatic action authority
No oracle framing
No trading signal authority
No unsafe escalation
Human review for sensitive states
Explicit fallback metadata
Runtime gate visibility
Permission Bits

X = execution permission
G = gate permission
O = oracle / prediction permission

Example:

X=0;G=0;O=0

Meaning:

execution blocked
gate escalation blocked
oracle / prediction authority blocked
